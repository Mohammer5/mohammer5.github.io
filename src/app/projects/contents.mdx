## [DHIS2 - Maintenance app rewrite](https://github.com/dhis2/maintenance-app-beta/pulls?q=is%3Apr+is%3Aclosed+author%3AMohammer5)

This project is a rewrite of the outdated [maintenance app] which is in such a
bad state that a gradual refactor is much more work than rewriting the app. On
this project I'm responsible for <u>managing the development team</u> and
<u>creating concrete tasks</u> from the collaboration between the research,
UI/UX and development teams that work in tandem on this project.

Albeit not seen by most DHIS2 users, this app is foundational as it allows
implementers to setup and configure the simple data types as well as more
complex structures that are used by almost every other app.

**Stack**: TypeScript, react, react-query, react-final-form, zod
